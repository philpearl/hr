
Not exactly useable by anyone, but controls the colour of a Hue light via an ANT+ HR monitor. The higher your 
heart rate the redder the light.

Needs libusb. `brew install libusb` did it for me.

Need to find the IP address of your Hue bridge, and to install a user on the bridge. See https://developers.meethue.com/develop/get-started-2/
